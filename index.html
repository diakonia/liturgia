DIAKONIA LITURGIA

This is the home page for Liturgia, a system for creating orders of service for Churches collaboratively online, including songs, liturgy, bible readings and media clips.

Liturgia creates files for OpenSong(tm). OpenSong is software for projecting worship materials

The Liturgia project is Open Source, licensed under the GPL v2 license - this means that anyone is free to use the software, to copy it and install it and pass it on to others, providing you do not try to charge for the software. More importantly, anyone is free to look at the source code for the project and amend it as needed, providing the software produced remains free and the amended source code is made freely available to everyone.
